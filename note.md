# PKu1664 放苹果 

> 2018 - 7 - 17

整数n的m划分数。

dp\[i][j] 表示 j 的 i 划分。

状态转移方程: dp\[i][j] = dp\[i - 1][j] + dp\[i][j - i]

这样保证了不会有重复计算，后一项表示了从 j - i 的 i 划分的每种情况中的每个数都加一，转移到 j 的 i 划分。
